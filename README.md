# RationalRage

A cartoon-style anonymous venting app split into a frontend and backend for easier hosting and deployment.

## Structure
- frontend: React + Vite + Tailwind UI
- backend: Express API

## Run locally
1. Install root tools:
   npm install
2. Start both services:
   npm run dev
3. Open the frontend at http://localhost:5173/
4. Backend health check at http://localhost:5000/api/health
